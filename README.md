# 介紹
slate 是類似 swagger 的 API 文件工具

😣 Windows 有無法開啟的 bug
![無法開啟的 bug](https://i.imgur.com/KWVM3UK.png)

```
// 安裝 ruby
sudo apt-get install ruby 
// 安裝 ruby-dev
sudo apt-get install ruby-dev
// 安裝 bundler
sudo gem install bundler
```

專案下執行(類似 npm install)
```
bundle install
```

啟動專案
```
bundle exec middleman server
```
